# Library Management System

## Запуск приложения

Запуск без seed-данных

```bash
go run main.go
```

## Запуск с автоматическим добавлением тестовых данных

```bash
go run main.go -seed
```

## Флаг -seed

Добавляет 5 тестовых книг и 2 тестовых пользователей
Срабатывает только если файлы books.txt и users.txt пустые
Предотвращает дублирование данных при повторных запусках
